# вҡҷпёҸ Installation

## 1. Clone the Repository

```bash
git clone https://github.com/your-username/weather-dashboard.git
```

## 2. Navigate to the Project Folder

```bash
cd weather-dashboard
```

## 3. Install Dependencies

```bash
npm install
```

## 4. Configure Environment Variables

Create a **.env** file and add your Weather API key:

```env
API_KEY=your_weather_api_key
PORT=3000
```

---

# в–¶пёҸ Run the Application



# рҹҢҗ API Workflow

1. User enters a city name.
2. The application sends a request to the Weather API.
3. Live weather data is retrieved.
4. The server processes the response.
5. Weather information is displayed dynamically on the dashboard.
